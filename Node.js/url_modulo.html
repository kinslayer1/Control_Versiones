Servidor de archivos Node.js

Llamar dos archivos solicitados por el usuario con el modulo URL

http://localhost:8080/summer.html
http://localhost:8080/winter.html

El modulo URL llama desde el servidor que es esste caso es el LOCALHOST:8080 mediante el archivo demo_fileserver.js a los archivos solicitados por el cleinte

summer.html
winter.html

Iniciar archivo demo_fileserver.js

Se debe inicializar el archivo demo_fileserver.js ubicandose en la ruta donde esta ubicado mediante el comando node demo_fileserver.js para que la maquina localhost funcione como un servidor escuchado por el puerto 8080

D:\A_A_JAIRO ROBLES\Visual studio code\Control_Versiones\Node.js>node demo_fileserver.js
